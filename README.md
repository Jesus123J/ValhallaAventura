# Camino al Valhalla ⚔️

Juego de **acción 2D con modo historia** de mitología nórdica, hecho en Unity. En desarrollo activo hacia una versión comercial (ver [Issues](../../issues) para el roadmap).

Eres **Björn Brazo-de-Hierro**: recorre el campo de los caídos, pelea con tu espada contra los draugr, encuentra la **Runa del Trueno** para desbloquear el **Rayo de Odín**, recoge almas y derrota al **Guardián** para abrir el portal.

## Características actuales

- 🗡️ Combate con espada, **dash** con estela e invulnerabilidad, y poder de rayo desbloqueable
- 🏆 Premios: runas de almas, pociones de hidromiel y runa de poder
- 👹 6 draugr + jefe guardián con corona
- 🌌 Nivel a color: aurora boreal, luna, montañas con parallax, pilares rúnicos, nieve
- 🎙️ Historia narrada con voz en español
- 🎵 Música y efectos sintetizados 100% por código
- 💾 Guardado automático de progreso (PlayerPrefs)
- 🧩 Todo el nivel, personajes y gráficos se generan por código (cero assets externos)

## Requisitos

- Unity **2022.3.42f1** (LTS)

## Cómo jugarlo

1. Clona el repo y agrégalo en Unity Hub (**Add project from disk**).
2. La escena `Aventura` se genera sola al abrir (menú **Herramientas → Crear Escena Aventura** si no).
3. **▶ Play**.

**Controles**: A/D correr · ESPACIO saltar · Clic/J espada · SHIFT dash · Q Rayo de Odín · 🔊 con volumen

## Roadmap

El plan de desarrollo hacia la versión vendible está en los **Issues** del repo, organizado en 5 fases: mecánica 3D/2.5D, progresión adictiva, contenido (capítulos y jefes), presentación, y lanzamiento comercial.
